# Personal Voice Profile Example

Use this file only as a schema for building a user's own local voice profile. Do not apply it as a default voice.

Create `local/personal-voice-profile.md` when the user wants a persistent private profile. Keep personal writing samples, names, unpublished materials, and company-specific style rules out of the public skill repository unless the user explicitly wants to publish them.
